# Todo List Manager (.NET 10)

A simple task management application built using C# 14 and .NET 10 that allows users to add, view, complete, and delete tasks. The app stores data in a JSON file, ensuring tasks persist between sessions. It includes features like due dates and color-coded overdue tasks, demonstrating core concepts such as CRUD operations, collections, file handling, and structured project design.

## Features
- Add, complete, delete tasks
- JSON file persistence
- Due date tracking
- Overdue highlighting

## Tech Stack
- C# 14
- .NET 10
- System.Text.Json
